# Anychef
Anychef is a lightweight recipe-finding tool that helps you discover delicious meals based on whatever ingredient you have at home. Simply enter a main ingredient, and Anychef fetches a list of recipes using the MealDB API. Built entirely with HTML, CSS, and JavaScript, this prototype is designed for quick, hassle-free meal inspiration.
